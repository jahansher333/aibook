## Course Structure

### 13 Weeks of Hands-On Learning

Each chapter follows a proven pedagogical structure:

1. **Learning Objectives** - Clear, measurable outcomes aligned with assessments
2. **Theory** - Conceptual foundations with diagrams and cited references
3. **Hands-on Lab** - Executable code examples for simulation and hardware
4. **Assessment** - Interactive MCQ quizzes to validate understanding

### Progressive Difficulty

- **Weeks 1-3**: Foundations (Physical AI concepts, ROS 2 basics, robot modeling)
- **Weeks 4-6**: Simulation (Gazebo, Unity, Isaac Sim environments)
- **Weeks 7-9**: Advanced AI (VLA models, kinematics, locomotion)
- **Weeks 10-12**: Integration (Manipulation, conversational AI, hardware deployment)
- **Week 13**: Capstone project (voice-command autonomous humanoid)
